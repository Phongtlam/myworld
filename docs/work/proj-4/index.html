Yap! is a voice-enabled, location specific search for tourists. The application was built with walking distance in mind and gives recommendations based on Google and Yelp APIs.

GitHub source code

Project overview:

  • Engineered using React.JS prioritize components’ reusability
  • Built using React Material UI library for better users’ experience
  • Node.JS for server side communications with Yelp and Google APIs
  • Data storage using MongoDB
